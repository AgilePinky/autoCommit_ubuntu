Это файл с краткой инструкцией по использованию скрипта для фиксации коммитов.

Описание для работы с файлом config.json и инструкция для получения API токена в джире будут добавлены позже.
Параметр jiraUrl - неполная ссылка до задачи в проекте Тикетон. Для завершения ссылки в форме скрипта добавляется идентификатор задачи.
Параметр usernameJira - логин в джире, впишите сюда свою почту по типу "i.sharipov@lantan.info".
Параметр usernameGrafana - логин в графане по типу "i.sharipov".
Параметр passwordGrafana - пароль в графане.
Параметры imagePath... - указывают на путь к картинке. Это устаревший функционал, который будет переработан, но в данный момент по прежнему используется логике скрипта.
Параметр TOKEN - api токен из джиры. Создается для каждого индивидуально и служит для авторизации при api запросах. Строго хранить и не выдавать его, как пароль. Есть возможность получить его повторно, инструкция получения токена далее.

Для получения токена api джиры необходимо действовать по следующей инструкции: Настроить аккаунт Jira > Безопасность > Создание токенов API и управление ими > Создать токен API
 > Ввести название токена > Выбрать дату завершения действия(в целях безопасности жизнь токена не более 1 года) > Создать.

 При необходимости удалить токен нажать кнопку "Отозвать" в списке ваших токенов.


Для запуска скрипта необходимо зайти в папку проекта и открыть терминал(или в команде прописать путь к папке проекта).
Далее вписать команду java -jar target/autoCommit-1.0-SNAPSHOT-jar-with-dependencies.jar для запуска скомпилированного jar файла со скриптом.
При необходимости снова скомпилировать jar файл после правок config.json ввести в папке проекта команду mvn clean package.


Для создания комментария со скриншотами нужных сервисов и веток необходимо вставить в поле ввода идентификатор задачи по типу "TDF-11111" и нажать на чек-боксы нужных сервисов на нужных ветках.
Далее с помощью кнопки "Сделать скриншот" скрипт открывает графану и делает в ней нужные скриншоты, сохраняя их в папке resources в проекте.
С помощью кнопки "Отправить комментарий" происходит отправка формы комментария со скриншотами через апи джиры.
