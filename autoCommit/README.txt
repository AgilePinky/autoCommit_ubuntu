Это файл с краткой инструкцией по использованию скрипта для фиксации коммитов.

Описание для работы с файлом config.json и инструкция для получения API токена в джире будут добавлены позже.

Для запуска скрипта необходимо зайти в папку проекта и открыть терминал(или в команде прописать путь к папке проекта). Далее вписать команду java -jar target/autoCommit-1.0-SNAPSHOT-jar-with-dependencies.jar для запуска скомпилированного jar файла со скриптом.

Для создания комментария со скриншотами нужных сервисов и веток необходимо вставить в поле ввода идентификатор задачи по типу "TDF-11111" и нажать на чек-боксы нужных сервисов на нужных ветках. Далее с помощью кнопки "Сделать скриншот" скрипт открывает графану и делает в ней нужные скриншоты, сохраняя их в папке resources в проекте. С помощью кнопки "Отправить комментарий" происходит отправка формы комментария со скриншотами через апи джиры.
